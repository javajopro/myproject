java, jsp, jqeury, oracle, tomcat, Html5, Css, 등을 사용하여 mvc패턴을 이용한
프로그래밍. 
1. 관리자, 사용자로 나뉜 로그인, 로그아웃 기능.
2. 관리자는 사용자의 정보를 열람 가능하며, 모든 댓글, 회원을 삭제 할 수 있다.
3. 모든 사용자는 게시판 열람 가능하며, 게시글 수정,삭제,삽입 파일 업로드가
가능하다.
4. 모든 사용자의 비밀번호를 암호화하여 보안성을 높힘.
5. mvc패턴을 통해 좀 더 직관적이고 가독성 있는 코드를 만드려고 함.
6. 다양한 쿼리를 이용하여 DB에도 좀 더 학습할 수 있도록 노력함.
